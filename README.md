# 太阳系运行模拟 (Solar System Simulation)

- 本项目所有（100%）代码均由 AI 生成。运行前请自行审查，任何后果概不负责。
- 一个基于 HTML5 Canvas 的交互式太阳系运行模拟程序，展示了太阳系八大行星的运行轨迹、速度对比以及视觉特效。

## 项目介绍

本模拟器是一个浏览器端的交互式天文模拟应用，展示了太阳系中八大行星的运行规律，包含以下功能：
- 实时模拟八大行星绕太阳运行
- 可调节的运行速度和缩放比例
- 轨道显示/隐藏切换
- 行星名称显示/隐藏切换
- 行星轨迹记录功能
- 暂停/继续模拟
- 点击行星查看详细信息
- 鼠标悬停高亮显示
- 精美的视觉效果（光晕、渐变、星星背景）

## 特性

- 🌟 **交互式控制面板** - 通过滑块调节速度和缩放
- 🎨 **精美视觉效果** - 行星光晕、渐变色彩、星星背景动画
- 📊 **详细信息面板** - 点击行星查看天文数据
- 🔄 **实时动画** - 平滑的动画和轨迹记录
- 🪐 **行星列表** - 右侧栏快速跳转至任意行星
- 🌙 **地球月球系统** - 地球附带月球运行动画

## 使用方法

1. **直接打开浏览器**：双击 `a/sun.html` 文件
2. **本地服务器运行**：使用任何 HTTP 服务器打开文件
   ```bash
   python -m http.server 8000
   ```
   然后访问 `http://localhost:8000/a/sun.html`

## 操作指南

### 控制面板
- **速度滑块**：调节行星运行速度（0-5倍）
- **缩放滑块**：放大/缩小整个模拟场景
- **显示轨道**：切换行星轨道线的显示
- **显示名称**：切换行星名称的显示
- **显示轨迹**：开启/关闭行星运行轨迹
- **暂停按钮**：暂停或继续模拟

### 交互操作
- **鼠标悬停**：高亮显示行星并显示信息
- **点击行星**：选中行星，显示详细信息
- **行星列表**：点击右侧列表中的行星快速跳转
